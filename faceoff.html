    How to play

    1. Five contestants line up. You'll see the reigning card with its value on the hidden measure.
    2. For each challenger, call whether it's higher or lower than the reigning card. The challenger then takes the throne.
    3. That's four calls in a row. Each gets a ribbon: right call wrong call
    4. Nail all four for a Clean Sweep. Values shown after each call — expect surprises.
